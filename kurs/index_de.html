Claude vergisst alles. Dieser Skill ändert das.

Normalerweise startet Claude jede Session bei Null. Deine Architekturentscheidungen, Bug-Fixes, Recherche-Ergebnisse — alles weg. Context Memory ändert das grundlegend.

05
Wissen hat ein Verfallsdatum — und eine Sicherheit

Nicht jede Information ist gleich sicher. Und „Umsatz 28 Mrd." und „Umsatz 34 Mrd." sind kein Widerspruch — sondern eine ZeitreiheJeder Node kann ein Quelldatum, ein Gültigkeitsfenster und eine Zeitkonfidenz tragen. So werden zeitliche Veränderungen sichtbar statt als Fehler behandelt..

Zeitdimension:
„KI-Adoption 12%" ← McKinsey, Q1 2025, Konfidenz: explicit
„KI-Adoption 23%" ← Accenture, Q4 2025, Konfidenz: explicit
→ Kein Widerspruch! Zeitreihe: +11 Prozentpunkte in 9 Monaten

Konfidenz:
[bug-002] Memory-Leak im Order-Service
  Relevanz: critical | Konfidenz: low
  → Wichtig, aber noch nicht bestätigt. Sobald bestätigt: Konfidenz → high
Wofür ist die Konfidenz-Angabe gut?
Sie bestimmt, wie weit oben ein Treffer in der Suche erscheint
Sie zeigt, ob eine Information bestätigt oder noch eine Vermutung ist
Sie legt fest, wie lange ein Wissensblock gespeichert bleibt
